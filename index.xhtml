pedantic presenter

This is a pedantic presenter

  • Write an (X)HTML5 document and call $.slides("rootelement")
  • The root's header and footer tags are on every slide
  • section tags become slides
  • Each section's header and footer show up, too
  • Simple keybindings for navigation: left/up/h/k/p/backspace go back, right/down/j/l/n/space go forward
  • See the next slide for an example

You can find/fork the code on github (BSD/MIT/X11 license):

http://github.com/wcmaier/pedantic-presenter/
